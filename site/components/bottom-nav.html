Bottom Nav

Mobile-only fixed bottom navigation bar with icon + label items. Hidden on desktop.

Preview

The bottom nav is only visible below 640px. Resize your browser or view on mobile to see it. The preview below simulates the mobile appearance.

App content area
Home Charts Log Predict Profile

Usage

Basic Usage

The bottom nav appears only on mobile (below 640px). Place it at the end of <body>. Add .rs-container to your main content wrapper to add bottom padding automatically.

HTML
<body>
    <main class="rs-container">
        <!-- App content -->
    </main>

    <nav class="rs-bottom-nav" aria-label="Main navigation">
        <a href="/" class="rs-bottom-nav__item rs-bottom-nav__item--active"
           aria-current="page">
            <svg width="20" height="20" viewBox="0 0 20 20"
                 fill="currentColor">
                <path d="M10.707 2.293a1 1 0 00-1.414 0l-7 7a1 1 0
                    001.414 1.414L4 10.414V17a1 1 0 001 1h2a1 1 0
                    001-1v-2a1 1 0 011-1h2a1 1 0 011 1v2a1 1 0 001
                    1h2a1 1 0 001-1v-6.586l.293.293a1 1 0
                    001.414-1.414l-7-7z"/>
            </svg>
            <span>Home</span>
        </a>
        <a href="/charts" class="rs-bottom-nav__item">
            <svg>…</svg>
            <span>Charts</span>
        </a>
        <a href="/log" class="rs-bottom-nav__item">
            <svg>…</svg>
            <span>Log</span>
        </a>
        <a href="/profile" class="rs-bottom-nav__item">
            <svg>…</svg>
            <span>Profile</span>
        </a>
    </nav>
</body>

SCSS Import

Import only the bottom nav component:

SCSS
@use "@rescale/scss/settings" as *;
@use "@rescale/scss/components/bottom-nav/bottom-nav";

Bottom Nav Classes

Class / AttrElementDescription
.rs-bottom-nav<nav>Container — hidden on desktop, fixed to bottom on mobile (≤640px)
.rs-bottom-nav__item<a>Nav item — flex column with icon + label, 52px min-width
.rs-bottom-nav__item--activeActive state — accent colour
.rs-container<main>Content wrapper — adds bottom padding on mobile to prevent overlap

Accessibility

  • ✓ Add aria-label="Main navigation" to the <nav> element
  • ✓ Mark the current page with aria-current="page" on the active link
  • ✓ Icons should have fill="currentColor" so colour changes with state
  • ✓ All tap targets meet the 44px minimum (52px min-width × 72px nav height)
